"My project is an Interrupt-Driven Automatic Visitor Counting System using the LPC2129 ARM7 microcontroller. Two IR sensors connected to EINT0 and EINT1 detect people entering and exiting the room. The microcontroller updates the visitor count in real time and displays it on a 20×4 LCD. It automatically turns the room light ON when at least one person is inside and OFF when the room becomes empty. A push button displays the total entered and exited counts, and hardware interrupts provide fast response with low CPU utilization."
